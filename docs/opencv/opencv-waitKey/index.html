opencv-waitKey

OpenCV 延迟操作 waitKey()#

经典操作

  • C++
  • Python
while(1) {
int key = waitKey(1);
if (char(key) == 27){
break;
}
}
while True:
key = cv2.waitKey(1)
if key == 27:
break

这里的 "27" 对应的是键盘对应按键的 ACSII 码值， 27 对应的是 ESC 键

  • 键盘所有按键对应的ASCII码值（键码值）
  • 实现按键交互
int key = waitKey(0);
switch (key)
{
case 't':
case 'T':
{
// code
}
break;
case 'f':
case 'F':
{
// code
} break;
case 'r':
case 'R':
{
// code
} break;
case 'c':
case 'C':
{
// code
} break;
case 27:
{
// code
} break;
}
