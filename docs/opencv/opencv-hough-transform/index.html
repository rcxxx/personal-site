OpenCV 中的霍夫变换

简介#

霍夫变换(Hough Transform)是图像处理中的一种特征提取技术，该过程在一个参数空间中通过计算累计结果的局部最大值得到一个符合该特征的集合作为霍夫变换的结果

霍夫线变换#

霍夫变换中最简单的是从二值图像中检测直线/线段，霍夫变换前，需要先检出图像边缘

霍夫变换的主要优点是能容忍特征边界描述中的间隙，并且相对不受图像噪声的影响

  • 一条直线在二维空间中可以由一个方程表示

在笛卡尔坐标系下，直线可以由 斜率 kkk 和 截距 bbb 表示

y=kx+by = kx + by=kx+b
  • 当直线垂直的时候，斜率不存在（无限大），kkk 的值有可能无限大，不利于计算，所以将直线转为参数方程表示

在极坐标下，直线可以由 极角 θ\thetaθ 和 极径 rrr 表示

y=(−cosθsinθ)x+(rsinθ)y = (-\frac{cos\theta}{sin\theta})x + (\frac{r}{sin\theta})y=(−sinθcosθ​)x+(sinθr​)

转换一下可以得到

r=xcosθ+ysinθr = xcos\theta + ysin\thetar=xcosθ+ysinθ

对于一个点 (x0,y0)(x_{0}, y_{0})(x0​,y0​) ，可以用 (r0,θ)(r_{0} , \theta)(r0​,θ) 表示通过这个点的所有的直线定义为

r0=x0cosθ+y0sinθr_{0}=x_{0}cos\theta+y_{0}sin\thetar0​=x0​cosθ+y0​sinθ
