什么是OpenCV

OpenCV 是一个跨平台计算机视觉的开源库，实现了很多图像处理和计算机视觉方面的通用算法

  • OpenCV官网

推荐几个很有趣的官方例程

  • Image Segmentation with Distance Transform and Watershed Algorithm

    • 这个例程将一堆杂乱堆放的扑克牌分割成独立的区域

  • Out-of-focus Deblur Filter

    • 这个例程对一张本来没对上焦的照片去模糊，将照片变得比较清晰
