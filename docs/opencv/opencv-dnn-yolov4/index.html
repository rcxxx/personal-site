使用 CV::DNN 模块读取 YOLO v4 权重进行实时目标检测

准备工作#

使用 DNN 模块需要编译 opencv_contrib

可以参考

  • [Install（仅CPU）]

如果需要开启 GPU 加速，可以参考

  • [Install（GPU）]

cmake 参数应添加下面这些

-DWITH_CUDA=ON \
-DOPENCV_DNN_CUDA=ON \
-DENABLE_FAST_MATH=1 \
-DCUDA_FAST_MATH=1 \
-DWITH_CUBLAS=1 \

在你有 GPU 的情况下这样可以启用 GPU 加速运算

DNN模块简单介绍#

下载文件#

先下载 YOLO v4 的配置文件和预训练的权重

  • yolov4.cfg
  • yolov4.weights

模型配置文件 yolov4.cfg 也可以在 opencv_extra 的 testdata/dnn 目录中找到

然后下载 coco_classes.txt 文件，因为预训练的权重使用的是 coco 数据集

  • coco_classes.txt
