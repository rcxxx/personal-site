## 记录
> <Highlight color="#CCCCCC"><b>不积跬步，无以至千里；不积小流，无以成江海。</b></Highlight>

- 知识库能使信息和知识有序化，增加效率
- 有效的梳理知识，建立知识体系
- 写笔记是一个好习惯
- 克服懒惰

## 分享
- 知识需要共享
- 与人交流、相互学习
- 闭门造车不是正确选项
